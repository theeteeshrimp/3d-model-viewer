# 🎲 3D Model Viewer

> *A web-based 3D model viewer powered by Three.js and React Three Fiber*

![Three.js](https://img.shields.io/badge/Three.js-black)
![React Three Fiber](https://img.shields.io/badge/React%20Three%20Fiber-blue)
![Next.js](https://img.shields.io/badge/Next.js-black)

## ✨ Features

- 🎨 **Load 3D Models** - Supports GLTF, GLB, OBJ formats
- 🖱️ **Interactive Controls** - Rotate, zoom, pan with mouse
- 📸 **Screenshot** - Export current view as PNG
- 🔧 **Display Options** - Wireframe mode, grid toggle
- 🎨 **Custom Background** - Choose your backdrop color

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/theeteeshrimp/3d-model-viewer.git
cd 3d-model-viewer

# Install
npm install

# Run
cd 3d-model-viewer
npm run dev

# Open http://localhost:3000
```

## 🎮 Controls

| Action | Control |
|--------|---------|
| Rotate | Left click + drag |
| Pan | Right click + drag |
| Zoom | Scroll wheel |

## 📁 Supported Formats

- GLTF (.gltf)
- GLB (.glb)
- OBJ (.obj) - partial support

## 🛠️ Tech Stack

- Next.js + TypeScript
- Three.js
- React Three Fiber
- React Three Drei
- Tailwind CSS
